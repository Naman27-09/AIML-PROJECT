# AIML-PROJECT

#STUDENT STRESS LEVEL PREDICTION SYSTEM

OVERVIEW

This project helps figure out if a student is stressed by looking at two things: how many hours they study and how much sleep they get. A computer program is used to make this prediction.

PROBLEM STATEMENT

Many students feel stressed because of much schoolwork and not doing healthy things every day. This project aims to find out if a student is stressed on using simple information.

SOLUTION

A special computer model called Logistic Regression is used to classify a students stress level into two groups:

* 0 → Not much stress

* 1 → A lot of stress

TECHNOLOGIES USED

* Python

* Pandas

* Scikit-learn

DATASET

A small set of information was made for this project. It includes:

* Study Hours

* Sleep Hours

* Stress Level

WORKING PROCESS

Here's how the project works:

* The information is. Stored using pandas.

* The study hours, sleep hours and stress level are separated.

* The information is split into two parts: one for teaching the computer and one for testing it.

* The computer is taught using the teaching part.

* The computer makes guesses using the testing part.
